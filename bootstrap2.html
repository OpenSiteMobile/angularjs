UI Bootstrap

Bootstrap components written in pure AngularJS by the AngularUI Team

This is a highly modified version of the page found at AngularJS-Bootstrap. It fully incorporates our MobileSiteOS version of AngularJS and MobileSiteOS module loading and control. It also uses our modularized version of Angular-Bootstrap. Thus, this page is a more complete example of how we use MobileSiteOS to control javascript functionality in a single page application. As demonstrated here, AngularJS can easily be integrated into this process with near identical (better?) results.

It is important to note that now the base page is very small. Page content and JavaScript (including AngularJS and application modules) are now being loaded on an "as needed" basis, so the initial page load is very fast, relative to the original page. Approximately one-third the initial load time. It also makes this SPA much cleaner and easier to maintain. Plus there is virtually no limit to the AngularJS controlled page content which could be injected and presented.
