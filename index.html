ngularJS Demos via MobileSiteOS

Here are some simple usage examples and test pages running AngularJS with MobileSiteOS. The first 8 simple demo's below are template content pages injected into this page, using a very basic technique for lazy-loading AngularJS code and content. They show the basics of using AngularJS, while demonstrating the flexibility of running them under MobileSiteOS.

The later are more complete "apps" demonstrating various levels of MobileSiteOS integration with AngularJS. Most are previously available demo's or pages which have been adapted to use MobileSiteOS for module loading, etc. We use these examples mostly for educational and testing purposes.
