Standard Version of AngularJS

{{greeting}}

This ultra simple AngularJS page is rendered using the standard version of AngularJS. This page does however load MobileSiteOS code too, as well as jQuery. MobileSiteOS console debugging output is available, but obviously our NgMomentum debugging output is not. While jQuery is not required specifically by AngularJS, we load it anyway in order to compare load times between the MobileSiteOS and Std. versions of AngularJS. This AngularJS page would be faster without them loading too.
